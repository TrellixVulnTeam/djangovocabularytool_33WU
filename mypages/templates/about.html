User guide



Instructions on how to use the chinese vocabulary tool

  1. In the home page, type enter a title (must be in english characters) for a new vocabulary set and press 'CREATE'.
  2. You can click 'View' button on that vocabulary set to be directed to the create page, where you can start adding vocabulary to the set.
  3. Type in the chinese vocabulary you want to add into the vocabulary set and press 'Save'.
  4. The vocabulary will be automatically generated in a new row, along with its phonetic writing and english translation.
  5. You can edit the english translation by clicking the 'Edit' button on that row.
  6. You can star and unstar vocabulary words by clicking the star icon on that row.
  7. You can delete the whole vocabulary row by clicking the 'Delete' button on that row.
  8. You can delete a whole vocabulary set by clicking the 'Delete' button at the top of the create page.
  9. Your vocabulary set will be automatically saved, and can be accessed through the home page (click the home icon to go to home).
  10. You can search for your vocabulary set or a particular word in the home page by clicking 'Search!'.
  11. You can export the finished vocabulary set to a PDF by clicking the 'Export set' button in the home or create page.

Sidenote: the translation is generated using Google Translate, and so may not be 100% accurate. Therefore the edit button allows you to change the english translation of a vocabulary word if you are not satisfied with the automatic translation provided.
