User guide



Instructions on how to use the chinese vocabulary tool

  1. In the home page, type in the chinese words you want to add into the vocabulary list and press 'Enter'.
  2. You will be redirect to another page where you will create the content of the vocabulary set.
  3. Type in the chinese vocabulary you want to add into the vocabulary list and press 'Enter'.
  4. The vocabulary will be automatically generated in a new row, along with its phonetic writing and english translation.
  5. You can edit the english translation by clicking the 'Edit' button
  6. You can delete the whole vocabulary row by clicking the 'Delete' button on that row
  7. Your vocabulary set will be automatically saved, and can be accessed through the home page
  8. You can export the finished vocabulary set to a PDF by clicking the 'Export set' button in the home or create page

Sidenote: the translation is generated using Google Translate, and so may not be 100% accurate. Therefore the edit function allows you to change the english translation if you are not satisfied with the automatic translation.
